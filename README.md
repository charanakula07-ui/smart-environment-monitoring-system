# Smart Environmental Monitoring System

## Overview
This project monitors environmental parameters like temperature, humidity, light, and motion using Raspberry Pi.

## Features
- Real-time monitoring
- Cloud integration using ThingSpeak
- CSV logging
- Alert system using LED and buzzer

## Technologies Used
- Python
- Raspberry Pi
- DHT11 Sensor
- LDR Sensor
- IR Sensor

## How to Run
1. Install required libraries
2. Connect sensors
3. Run:
   python3 smart_monitor.py
